## 课堂笔记

```sql
select * from users where id=1

-- 
select * from users where id='1 and 1=1'; 
```

**当ID为type 为数字时， 却传入字符串时 。 mysql 将会将字符串进行强转为数字， 知道出现第一个数字为止**
+ **首字母** 为数字时， 则为数字
    + `23adfas` => 23
    + `3sfadf` => 0
+ **字母** 为字符时， 则为 0， 并停止后面的转换。
